# Thincrust

A minimal Next.js application with Tailwind CSS.

## Features

-   Next.js 15
-   React 19
-   Tailwind CSS
-   TypeScript

## Getting Started

```bash
# Install dependencies
npm install

# Run the development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
